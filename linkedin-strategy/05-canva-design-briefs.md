## VISUAL 1: LinkedIn Personal Banner

**Dimensions:** 1584 x 396 px
**Canva:** Create a design → Custom size → 1584 x 396

### Design Brief:

```
BACKGROUND:
- Gradient from left to right: #0A0A0A → #28208C → #0A0A0A
- OR: Search Canva templates "technology dark banner linkedin" and pick one with dark blue/purple tones

ELEMENTS:
- Subtle dot grid or circuit board pattern at 10-15% opacity (search "tech pattern" in Canva elements)
- Small abstract geometric shapes (circles, hexagons) in #7065EF at 20% opacity scattered across

TEXT (right-aligned, 40% from right edge):
- Line 1: "AI  |  Cloud  |  Impact" in Montserrat Bold, #E9E9E9, ~28px
- Line 2: (none — keep it clean and minimal)

OPTIONAL:
- Very subtle MEYNG logo watermark bottom-right at 15% opacity

DO NOT:
- Add your photo (LinkedIn already shows it)
- Add too much text (it competes with your profile info below)
- Use bright/neon colors
```

### Canva Steps:

1. New design → Custom 1584 x 396
2. Background: Click background → Color → #0A0A0A
3. Add rectangle full-width: gradient #28208C to #0A0A0A (use Canva gradient feature)
4. Elements → Search "circuit pattern" or "tech dots" → Add at low opacity
5. Text → "AI | Cloud | Impact" → Montserrat Bold → White → Right-align
6. Download as PNG

## BATCH CREATION WORKFLOW

To create all visuals efficiently in one Canva session:

1. **Create brand kit first** (colors + fonts)
2. **Start with the LinkedIn banners** (personal + company) — these are one-time
3. **Create one "master post template"**: 1200x627, dark bg, Montserrat font, logo positioned — then duplicate for each post
4. **For each post**: Duplicate master → change text/icons → export
5. **Name files clearly**: `linkedin-post-01-launch.png`, `linkedin-post-02-sangoai.png`, etc.
6. **Export all as PNG** at highest quality

Estimated time: 2-3 hours for all 13 visuals (faster after the first 2-3).
